## 🖥️ Funcionalidades

- **Página principal**: vitrine de produtos, categorias e destaques.
- **Menu responsivo**: navegação adaptada para desktop e mobile.
- **Seção Hero**: destaque visual com chamada para ação.
- **Categorias**: navegação rápida por estilos de tênis.
- **Grid de produtos**: exibição de modelos em destaque.
- **Rodapé**: links institucionais, redes sociais e newsletter.

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3 (com organização modular por componentes)
- Imagens otimizadas para web

## 📱 Responsividade

O site é responsivo, adaptando-se a diferentes tamanhos de tela (desktop, tablet, mobile). As imagens e o layout são ajustados via media queries para garantir boa experiência em todos os dispositivos.

## 🚀 Como rodar o projeto

1. Clone ou baixe este repositório.
2. Abra o arquivo `https://eleilson-lil.github.io/ecommerce-syntaxwear/` em seu navegador.
3. Não é necessário backend ou instalação de dependências.

## ✨ Customização

- Para alterar imagens, substitua os arquivos na pasta `images/`.
- Para modificar estilos, edite os arquivos em `css/` e `css/components/`.
- Para adicionar produtos ou categorias, edite o `index.html`.
